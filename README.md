# hatch-messaging-service

how to run:

The maven dependencies are in the mvnw exe so you should be able to run as long as your IDE has a jdk 17+. Make sure to compile the pom.xml with all the dependencies as well.

`make setup`

`make db-up`

`make run`

New Terminal run 

`make test`

`make db-shell`

Cleanup

`make db-down`

`make cleanup`
